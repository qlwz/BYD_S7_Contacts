# BYD S7 通讯录同步工具

工具使用方法：
    1、手机安装【QQ同步助手】
    2、登录并同步通讯录
    3、打开【http://ic.qq.com】并登录
    4、找到：更多操作 -> 导出联系人到本地 -> CSV -> 导出
    5、使用该工具生成数据库
    6、把目录下的【GPS_BYD】目录复制到TF卡中
    7、将TF卡插入S7的MAP插槽，按【导航】即可完成同步


蓝牙名称 查看方法：
    Android查看方法：设置 -> 蓝牙 -> 开启蓝牙 -> 名称
    iPhone查看方法：设置 -> 通用 -> 关于本机 -> 名称

    如不能正常显示，请查看车机中蓝牙设备名称。


分组 说明：
    QQ同步中可以建立分组：
    1、不常用联系人   该分组的联系人不同步
    2、常用联系人     该分组的联系人名称补充0（优先显示）
